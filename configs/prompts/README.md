# Prompts Config Directory

This directory is reserved for LLM evaluation prompt templates, prompt judge instructions, and task templates.

## Structure
- `/configs/prompts/judge_templates.yaml`: Holds templates for LLM-as-a-judge prompts.
- `/configs/prompts/eval_tasks.yaml`: Holds specific dataset-to-prompt templates for ROUGE, Exact Match, and BERTScore runs.
